# ⏱️ Stopwatch Web Application

A modern and responsive Stopwatch Web Application built using HTML, CSS, and JavaScript as part of my Web Development Internship at Prodigy InfoTech.

## 🚀 Features

- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Lap Time Tracking
- Responsive Design
- Smooth User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📚 What I Learned

While building this project, I improved my understanding of:

- DOM Manipulation
- JavaScript Timing Functions
- Event Handling
- Responsive Web Design
- Managing Real-Time UI Updates
